שירות קטן לניהול משימות ב-Java
--------------------------------

תיאור
פרויקט לדוגמה שמיישם שירות פשוט לניהול משימות (קריאה/כתיבה ל-JSON).

דרישות
ניהול גרסאות git
שימוש ב־Java סטנדרטי בלבד (ללא ספריות חיצוניות).

התקנה והרצה
1. התקנת הפרויקט:
    git clone https://github.com/syankel/JavaTest

2. לפתוח את סביבת העבודה (VS Code) ולהתאים את נתיב קובץ ה-JSON במידה ונדרש:
    הקובץ שמכיל את הנתונים: PartOne/Task_JSON.json
   התוכנית קוראת קובץ JSON ממיקום יחסי/מוחלט, יש לעדכן את הנתיב בקוד במקום בו מוגדר הנתיב לקובץ 
   אם הקובץ ריק, יש להזין את דוגמת המבנה שמופיע מטה 

3. מעבר לתקיית PartOne, קומפילציה והרצה:
    cd PartOne
    javac *.java
    java Main


דוגמת מבנה JSON (Task_JSON.json)
[
  {
     "id": 1,
     "title": "משימה לדוגמה",
     "description": "תיאור המשימה",
     "status": false
  }
]
